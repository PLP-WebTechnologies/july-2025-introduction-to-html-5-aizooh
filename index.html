1. What is Semantic HTML5?

Semantic HTML refers to the use of tags that convey the **meaning** of the content, not just how it should look. Instead of using generic tags like `div` everywhere, we use tags like `header`, `nav`, `main`, `section`, `article`, and `footer`.

This approach improves machine readability for search engines and assistive technologies like screen readers.
